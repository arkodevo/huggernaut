# Formula Guidelines — 流動 Living Lexicon

**Version:** 1.0
**Date:** 2026-04-03
**Authors:** Louie + Guangliu

---

## Purpose

Formulas show learners **how a word fits into a sentence**. They are structural patterns, not definitions. A good formula teaches argument structure, word order, and common frames at a glance.

---

## Notation

### Brackets `[ ]` — Variable slots

Square brackets mark **slots the learner fills in**. The label inside describes what goes there.

```
買 + [Object]
很 + [Stative Word]
告訴 + [Person] + [Thing/Clause]
```

**Standard slot labels:**
- `[Verb]` — any verb or verb phrase
- `[Noun]` — any noun or noun phrase
- `[Modifier]` — anything that modifies (stative verbs, clauses, adjective-like words)
- `[Clause]` — a full clause or sentence
- `[Person]`, `[Place]`, `[Food]`, `[Clothing]`, etc. — semantic specifics when helpful
- `[Number]` — a number or quantity

**Do NOT use:**
- `[Adj]` — we don't have "adjectives"; stative verbs (Vs/Vst/Vs-attr) fill this role. Use `[Modifier]` instead.
- `[Stative Word]` — too technical. Use `[Modifier]` instead.
- POS slugs as slot labels (`[V]`, `[Vi]`, `[Vst]`) — these are internal, not learner-facing.

### Plus `+` — Concatenation

The `+` sign connects elements in sequence. Spaces around `+` for readability.

```
跟 + [Person] + 說話
[Person] + 很 + 累
```

### Slash `/` — Alternative patterns

The `/` separates **alternative formulas** for the same word. Each side is a complete pattern.

```
喝 + [Drink] / 喝了
準備 + [Thing] / 準備 + [Verb Phrase]
```

### No brackets — Fixed elements

Words without brackets appear as-is. These are the constants in the pattern.

```
先 + [Clause]，然後 + [Clause]
```

Here `先`, `然後`, and the comma are fixed; the clauses are variable.

## Patterns by POS

### Nouns (N)

Show the most natural frame — usually with a measure word or possessive.

```
[Possessor] + 姊姊
[Number] + 封 + 信
[Number] + 張 + 床
```

Do NOT use bare measure words as the formula (e.g., `一個X` is not useful). Show the pattern with variable slots.

### Pronouns (Prn)

Show subject and object positions.

```
我 + [Verb/Phrase] / [Verb] + 我
```

### Verbs — Transitive (V)

Show `V + [Object]` with a specific object type.

```
穿 + [Clothing/Shoes]
烤 + [Food/Thing]
送 + [Object/Person] + 給/到 + [Target]
```

For ditransitive verbs, show both objects:

```
告訴 + [Person] + [Thing/Clause]
教 + [Person] + [Thing]
```

### Verbs — Intransitive (Vi)

Show the word in its natural frame. No object slot.

```
跑 / 跑得 + [Complement]
哭 / 哭了
旅行 / 去 + [Place] + 旅行
```

### Verbs — Separable (Vsep)

Show BOTH the joined form and the split form. This is the key teaching moment.

```
聊天 / 跟 + [Person] + 聊天
幫忙 / 幫 + [Person] + 忙
唱歌 / 跟 + [Person] + 唱歌
洗澡 / 洗了個澡
```

### Verbs — Stative (Vs)

Show the 很-pattern as the default frame.

```
[Thing/Person] + 很 + 貴
[Person] + 很 + 聰明
```

For Vs words with special patterns, show those:

```
差不多 / [Quantity/Time] + 差不多
小心！ / 小心 + [Clause]
```

### Verbs — State-Transitive (Vst)

Show the word with its complement/object.

```
喜歡 + [Noun/Verb Phrase]
知道 + [Object/Clause]
覺得 + [Clause/Phrase]
```

### Verbs — Process (Vp, Vpt)

Show the state change or result pattern.

```
壞了 / [Thing] + 壞了          (Vp — intransitive)
發現 + [Thing/Problem/Clause]   (Vpt — transitive)
```

### Verbs — Auxiliary (Vaux)

Show `Vaux + [Verb]`.

```
能 + [Verb]
應該 + [Verb/Phrase]
可能 + [Verb/Clause]
```

### Adverbs (Adv)

Show `Adv + [Verb/Phrase]` or the frame it occupies.

```
很 + [Stative Word]
都 + [Verb/Phrase] — with plural topic: [Plural Topic] + 都 + [Verb/Phrase]
已經 + [Verb/Phrase]
```

### Conjunctions (Conj)

Show the two clauses/elements connected. Use `[A]`/`[B]` or `[Clause]`.

```
[A] + 和 + [B]
[Clause]，可是 [Clause]
如果 + [Clause]，就 + [Clause]
```

### Prepositions (Prep)

Show the prepositional phrase in context.

```
從 + [Place/Time] + [Verb Phrase]
在 + [Place] + [Verb Phrase]
比 + [Noun/Pronoun] + [Stative Word]
```

### Determiners (Det)

Show the determiner with its noun.

```
這 + [Measure Word] + [Noun]
每 + [Measure Word] + [Noun]
```

### Measure Words (M)

Show `[Number] + M + [Noun]`.

```
[Number] + 個 + [Noun]
[Number] + 張 + [Flat Object]
```

### Particles (Ptc)

Show the particle in its grammatical position.

```
[Verb] + 了
[Verb] + 過
[Clause] + 嗎？
```

### Resultative Complements (Vcomp)

Show `V + Complement` pattern with example verbs.

```
V + 住 (記住, 抓住, 擋住)
V + 見 (看見, 聽見, 碰見)
V + 好 (做好, 準備好, 想好)
```

## Fixed Expressions

Some words ARE the formula — they have no variable structure. Write them as-is:

```
再見
對不起
不客氣。
沒關係
```

Add a period/punctuation only if the expression is always sentence-final.

---

## Rules

1. **The word MUST appear in its own formula.** No exceptions.
2. **Use Chinese in formulas**, not English. Slot labels in `[ ]` are the only English.
3. **Keep it short.** One or two patterns maximum. If there are many patterns, pick the two most useful.
4. **Be specific about object types** when it helps: `[Food]` is better than `[Object]` for 烤.
5. **Do NOT include separable notes** like `(V-O: 走+路)`. The POS chip already says Vsep.
6. **Do NOT write definitions as formulas.** `to walk` is not a formula. `走路 / 走路去 + [Place]` is.
7. **Formulas are for learners.** Prioritize the patterns a learner needs to produce correct sentences.
